## Kaleido Path
- Mantém os controles existentes: Symmetry, Brush, Color, Clear, Generate e Export PNG.
- Novo painel RECORD MOTION:
  - record: arma gravação; desenhe no canvas; ao soltar o mouse, o movimento toca automaticamente.
  - erase: apaga a gravação e limpa o canvas.
  - play speed: controla velocidade de reprodução.
  - stroke length: controla a quantidade de traços reproduzidos por etapa.
